CSS Positioning

This page demonstrates different CSS positioning techniques

Static - can not be regulated with styles,
top, bottom, left, right and z-styles
Relativ - can not be regulated with styles,
top, bottom, left, right and z-styles
Relative - can be regulated with styles,
top, bottom, left, right and z-styles
Absolute - can be regulated with styles,
top, bottom, left, right and z-styles
Absolute - can be regulated with styles,
top, bottom, left, right and z-styles
Fixed positioning stays always fixed,
independent from scrolling
Sticky positioning can be regulated with styles,
top, bottom, left, right and z-index styles
